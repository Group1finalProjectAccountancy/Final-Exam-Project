## 📋 Tentang Project

Karawang selama ini dikenal sebagai kota industri terbesar di Indonesia. Namun di balik citra industrialnya, Karawang menyimpan potensi wisata yang luar biasa — dari pantai tersembunyi, situs sejarah Proklamasi, candi Buddha tertua di Indonesia, hingga kekayaan kuliner khas Sunda pesisir.

Project ini hadir sebagai **panduan wisata digital** yang memanfaatkan teknologi informasi untuk memperkenalkan destinasi wisata Karawang kepada masyarakat luas, khususnya generasi muda.

## 🚀 Cara Deploy ke GitHub Pages

### Langkah 1 — Clone / Upload Repository
```bash
git clone https://github.com/username/explore-karawang.git
```

### Langkah 2 — Masuk ke folder project
```bash
cd explore-karawang
```

### Langkah 3 — Push ke GitHub
```bash
git add .
git commit -m "Initial commit: Explore Karawang website"
git push origin main
```

### Langkah 4 — Aktifkan GitHub Pages
1. Buka repository di GitHub
2. Klik **Settings** → **Pages**
3. Pada *Source*, pilih **Deploy from a branch**
4. Pilih branch **main** → folder **/ (root)**
5. Klik **Save**
6. Website live di: `https://username.github.io/explore-karawang`
